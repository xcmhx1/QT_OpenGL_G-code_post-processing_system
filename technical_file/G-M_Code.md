# CNC 加工中心 G/M 代码及地址码详解指南

## 一、 G 代码（准备功能）

G 代码决定了机床的运动模式和逻辑状态。

### 1. 基本运动指令

| 代码    | 功能           | 格式                     | 说明                                              |
| ------- | -------------- | ------------------------ | ------------------------------------------------- |
| **G00** | 快速定位       | G00 X_ Y_ Z_             | 刀具以最高速度移动，不进行切削。                  |
| **G01** | 直线插补       | G01 X_ Y_ Z_ F_          | 按指定的进给速度 F 进行直线切削。                 |
| **G02** | 顺时针圆弧插补 | G02 X_ Y_ Z_ I_ J_ K_ F_ | I/J/K 为圆心相对起点的偏移量（或用 R 指定半径）。 |
| **G03** | 逆时针圆弧插补 | G03 X_ Y_ Z_ I_ J_ K_ F_ | 同上，方向为逆时针。                              |
| **G04** | 暂停/准停      | G04 P_ 或 G04 X_         | 在指定时间内停止进给（P 毫秒，X 秒）。            |

### 2. 平面选择与坐标系

| 代码          | 功能           | 说明                               |
| ------------- | -------------- | ---------------------------------- |
| **G17**       | XY 平面选择    | 默认平面，激光加工通常使用此平面。 |
| **G18**       | XZ 平面选择    | 常用于车床或特殊侧面加工。         |
| **G19**       | YZ 平面选择    | 常用于侧面加工。                   |
| **G15**       | 极坐标取消     | 退出极坐标编程模式。               |
| **G16**       | 极坐标有效     | 使用极坐标（半径与角度）进行编程。 |
| **G54 - G59** | 工件坐标系选择 | 选择预设的 1 至 6 号工件坐标系。   |

### 3. 单位与编程模式

| 代码    | 功能         | 说明                                   |
| ------- | ------------ | -------------------------------------- |
| **G20** | 英寸输入     | 使用英寸作为长度单位。                 |
| **G21** | 毫米输入     | 使用毫米作为长度单位（激光加工常用）。 |
| **G90** | 绝对坐标编程 | 坐标值相对于工件原点。                 |
| **G91** | 增量坐标编程 | 坐标值相对于当前位置。                 |
| **G94** | 每分钟进给   | F 单位为 mm/min 或 inch/min。          |
| **G95** | 每转进给     | F 单位为 mm/r（激光加工通常不用）。    |

### 4. 刀具补偿

| 代码    | 功能           | 说明                             |
| ------- | -------------- | -------------------------------- |
| **G40** | 半径补偿取消   | 停止半径偏置。                   |
| **G41** | 刀具半径左补偿 | 沿运动方向看，刀具偏在工件左侧。 |
| **G42** | 刀具半径右补偿 | 沿运动方向看，刀具偏在工件右侧。 |
| **G43** | 长度正补偿     | 考虑刀具长度，向上偏置。         |
| **G44** | 长度负补偿     | 考虑刀具长度，向下偏置。         |
| **G49** | 长度补偿取消   | 取消长度偏置。                   |

### 5. 固定循环指令（孔加工）

| 代码    | 功能           | 常用格式                                 |
| ------- | -------------- | ---------------------------------------- |
| **G73** | 高速深孔啄钻   | G73 X_ Y_ Z_ R_ Q_ F_ (Q 为每次切入深度) |
| **G80** | 固定循环取消   | 停止当前的钻孔/攻丝循环。                |
| **G81** | 浅孔钻循环     | 一次性钻到底。                           |
| **G83** | 深孔排屑钻循环 | 每次钻深后退回 R 点排屑。                |
| **G84** | 刚性攻丝       | 用于加工内螺纹。                         |
| **G98** | 返回初始平面   | 固定循环结束后回到起始点高度。           |
| **G99** | 返回 R 点平面  | 固定循环结束后回到 R 点高度。            |

## 二、 M 代码（辅助功能）

M 代码负责开关控制（主轴、冷却液等）。

| 代码    | 功能           | 说明                                   |
| ------- | -------------- | -------------------------------------- |
| **M00** | 程序无条件停止 | 用于测量或手动换向。                   |
| **M01** | 选择性停止     | 仅在机床面板“选择停止”开关打开时生效。 |
| **M02** | 程序结束       | 程序停止，光标停在末尾。               |
| **M03** | 主轴正转       | 在激光 CAM 中常映射为“开光”。          |
| **M04** | 主轴反转       | 激光加工较少使用。                     |
| **M05** | 主轴停止       | 在激光 CAM 中常映射为“关光”。          |
| **M06** | 自动换刀       | 执行换刀动作。                         |
| **M08** | 冷却液开       | 在激光加工中常映射为“吹气”。           |
| **M09** | 冷却液关       | 停止吹气。                             |
| **M30** | 程序结束并复位 | 运行结束，光标回到程序开头。           |
| **M98** | 调用子程序     | M98 P_ (P 后跟子程序号)。              |
| **M99** | 子程序结束     | 返回主程序。                           |

## 三、 地址码（英文字母）含义

地址码定义了 G 代码后参数的具体意义。

| 字母          | 含义                | 在 CAM 开发中的应用提示                           |
| ------------- | ------------------- | ------------------------------------------------- |
| **X / Y / Z** | 坐标轴              | 指定终点的绝对或相对位置。                        |
| **I / J / K** | 圆心偏移量          | I 对应 X 轴偏移，J 对应 Y 轴，K 对应 Z 轴。       |
| **R**         | 圆弧半径 / 返回点   | 在 G02/G03 中代表半径；在固定循环中代表安全高度。 |
| **F**         | 进给速度            | 决定切削/切割的移动快慢。                         |
| **S**         | 主轴转速            | 在激光中代表 **激光功率**。                       |
| **T**         | 刀具号              | 用于选择不同的激光头或加工参数组。                |
| **D / H**     | 补偿号              | 指定存储在机床寄存器里的半径/长度补偿值。         |
| **P**         | 暂停时间 / 子程序号 | 在 G04 中代表时间，在 M98 中代表程序号。          |
| **Q**         | 深度 / 偏移量       | 在深孔加工中代表每次钻入的深度。                  |
| **L**         | 重复次数            | 用于循环或子程序的多次执行。                      |
| **N**         | 程序段号            | 用于标记代码行号，方便搜索和跳转。                |
| **O**         | 程序编号            | 整个程序的标识符（如 O0001）。                    |
